## 主界面

![avatar](/images/mainui.png)

编辑器主界面由以下几个部分组成：

* 菜单栏。主要操作区。
* 工具栏。常用的功能按钮。
* 侧边栏。显示列表、搜索、新建等。
* 图谱区。关系图谱自动绘制区域。
* 状态栏。显示控制台输出的最后一条信息。
